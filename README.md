# File Transfer System (Python)

A simple file transfer system built using Python and Flask that allows users to send files over the internet using a generated link.

## Features

- Upload files through a web interface
- Receive files directly on the system
- Progress tracking during upload
- Username-based file organization
- Basic duplicate file handling

## Tech Stack

- Python (Flask)
- HTML, CSS, JavaScript
- Local file system storage

## How It Works

1. User generates a link
2. Shares the link with another user
3. The other user uploads the file
4. File is received and stored on the system
